# Bazaar IITGN

A campus marketplace built for the IIT Gandhinagar community where students can buy, sell, and exchange items with other students.

The idea behind this project is to make it easier for students to find second-hand items such as books, electronics, hostel essentials, and other everyday products without relying on external marketplaces. Everything is designed around a campus environment, making transactions simpler and more trustworthy.

---

## Features

* User authentication
* Create, edit and delete listings
* Browse available products
* Search items
* Watchlist for saving interesting listings
* Chat between buyers and sellers
* Make and manage offers
* User reputation and reviews
* Report inappropriate listings
* Admin dashboard for moderation
